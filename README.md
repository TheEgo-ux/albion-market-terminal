# Albion Market Terminal

Herramienta todo-en-uno para el mercado de **Albion Online**. Precios en vivo, flipping, rutas comerciales y calculadora de crafteo con datos de [Albion Data Project](https://www.albion-online-data.com/).

## Funcionalidades

- **Precios Hoy** — Consulta precio de compra/venta de cualquier ítem en todas las ciudades con detección de arbitraje.
- **Flipping** — Compra ítems planos (.0), encántalos y véndelos en el Mercado Negro. Calcula profit con setup fee del 2.5%.
- **Rutas** — Encuentra rutas rentables entre ciudades con cálculo de impuesto, peso y montura óptima.
- **Crafteo** — Calcula costos reales de producción con RRA, foco, bonos de ciudad y break-even.

## Cómo usar

1. Abre `albion_market_terminal_v7.html` en cualquier navegador (no requiere instalación).
2. Selecciona tu servidor (Americas / Europa / Asia).
3. Navega entre las pestañas inferiores: Precios, Flipping, Rutas, Crafteo.

## Tecnologías

- HTML / CSS / JavaScript vanilla (archivo único, sin dependencias externas)
- API de [Albion Data Project](https://www.albion-online-data.com/) para precios en vivo
- Diseño responsive con modo oscuro
